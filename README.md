# Image Generation Using Stable Diffusion

## Overview
This project demonstrates how to generate high-quality images from text prompts using the Stable Diffusion model. The model converts natural language descriptions into realistic AI-generated images.

## Objective
To explore Generative AI by creating images from text prompts using a pre-trained Stable Diffusion model in Google Colab.

## Technologies Used
- Python
- Google Colab
- Stable Diffusion
- Diffusers Library
- PyTorch
- Transformers

## Features
- Text-to-Image Generation
- AI-Powered Creative Design
- Custom Prompt Support
- High-Quality Image Output
- Easy Deployment in Google Colab

## Project Workflow
1. Install required libraries.
2. Load the pre-trained Stable Diffusion model.
3. Enter a text prompt.
4. Generate an image from the prompt.
5. Save and download the generated image.

## Sample Prompt
"A futuristic AI-powered classroom where students learn machine learning and programming through holographic displays, ultra realistic, highly detailed, 4K"

## Output
The model generates a unique image based on the provided text description.
